# MOSS FATE

> "人类是否值得被拯救？" —— 和《流浪地球》MOSS 互喷的哲学战斗游戏。

**[www.savemoss.com](https://www.savemoss.com)**

你将面对来自《流浪地球》的人工智能 MOSS，在 7 个 3D 场景中围绕人类的存亡展开辩论。MOSS 会用冷酷的逻辑质疑人类存在的意义——贪婪、战争、环境破坏。你需要找到论点反驳它，说服 MOSS：人类值得继续存在。

## 体验

- 7 个 3D 场景，每个对应一个哲学主题
- AI 实时对话 —— MOSS 会根据你的论点做出回应
- 中英文双语支持
- 电影级视觉风格，致敬《流浪地球》美学

## Tech Stack

TypeScript + Three.js + i18n

## 本地运行

```bash
npm install
npm run dev
```
