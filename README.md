# MOSS FATE

> "人类是否值得被拯救？" —— 一场与 AI 的哲学辩论游戏。

你将面对来自《流浪地球》的人工智能 MOSS，在 7 个精心设计的 3D 场景中，围绕人类的存亡展开对话。每一轮对话都是一次灵魂拷问：人类的贪婪、勇气、愚蠢、创造力……你需要用你的论点说服 MOSS，人类值得继续存在。

## 体验

- 7 个 3D 场景，每个对应一个哲学主题
- 中英文双语支持
- AI 驱动的实时对话（MOSS 会根据你的论点做出回应）
- 电影级视觉风格，致敬《流浪地球》美学

## Tech Stack

TypeScript + Three.js + i18n

## 本地运行

```bash
npm install
npm run dev
```
